# MediConnect Smart Community Health Management System (SCHMS)

## Project Overview
The MediConnect Smart Community Health Management System (SCHMS) is a digital healthcare management solution designed to improve healthcare service delivery by replacing manual record-keeping with an efficient electronic system.

The system enables healthcare facilities to manage patient records, appointments, diagnoses, prescriptions, and reports in one centralized platform. It aims to improve efficiency, reduce paperwork, enhance patient care, and support informed decision-making.

## Project Objectives
- Digitize patient registration.
- Enable appointment scheduling.
- Store electronic medical records.
- Record patient diagnoses.
- Manage treatment prescriptions.
- Generate healthcare reports.
- Improve healthcare service delivery.

## Features
- Patient Registration
- Secure User Login
- Appointment Booking
- Patient Dashboard
- Health Worker Dashboard
- Diagnosis Management
- Treatment Prescription
- Medical Records Management
- Administrative Dashboard
- Report Generation
